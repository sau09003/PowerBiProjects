# 📱 Mobile Sales Analysis Dashboard | Power BI Project

## 📌 Overview

An interactive Power BI dashboard developed to analyze mobile sales performance, customer behavior, product trends, and regional sales insights.

This project demonstrates the complete data analytics workflow:
- Data Preparation
- Data Transformation
- DAX Calculations
- Data Visualization
- Business Insights Generation
